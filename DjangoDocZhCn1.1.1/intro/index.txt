    因为 Django 是 100% 基于 Python 的。所以你是 Python_ 新手，那么你最好先学习
    Python 。如果没有坚实的 Python 基础，是不可能用好 Django 的。
    
    如果你是编程新手，那么你最好先看看 `编程新手 Python 资源列表`_ 。
    
    如果你有其他语言的编程经验并想快速学习 Python ，那么我们推荐
    `Dive Into Python`_ （这里有 `dead-tree 版本`_ 和 `中文版`_ ）。如果这本书不符合你的
    要求，这里还有许多其他 `关于 Python 的书`_ 。

    .. _中文版: http://www.woodpecker.org.cn/diveintopython/
    .. _dead-tree 版本: http://www.amazon.com/exec/obidos/ASIN/1590593561/ref=nosim/jacobian20
    .. _关于 Python 的书: http://wiki.python.org/moin/PythonBooks
